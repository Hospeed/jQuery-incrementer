jQuery-incrementer
==================

This plugin extends an html input field which serves a number to be given.
Handling validation and easy incrementing or decrementing of given value is
provided.

Examples:
---------

```javaScript
$('input').Incrementer();
```

<input type="text"></input>
<script type="text/javascript">
    $('input').Incrementer();
</script>
